# IOG — Idle Or Going?

> 나는 지금 노는 중인가, 뭔가 하고 있는 중인가?

**IOG**는 Windows PC에서 백그라운드로 조용히 동작하며 내가 어떤 프로그램을 얼마나 쓰고 있는지 자동으로 기록하고, 매일의 사용 패턴을 보고서로 보여주는 개인용 자기관찰(self-tracking) 도구입니다.

매번 시작/종료 버튼을 눌러야 하는 타임트래킹 도구 대신, **그냥 켜두면 자동으로 쌓이는** 가벼운 기록을 지향합니다.

## Why IOG?

하루를 돌아보면 "오늘 뭐 했지?"라는 질문에 막상 답하기 어려운 순간이 많습니다. IOG는 그 질문에 데이터로 답합니다 — 내가 의식하지 않아도 기록은 계속 쌓이고, 하루가 끝나면 무엇에 시간을 썼는지 보여줍니다.
